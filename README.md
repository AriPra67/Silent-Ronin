# Silent Ronin

## Overview

Silent Ronin is a 2D action platformer game developed in Unity. Players take control of a samurai warrior and progress through levels while avoiding hazards, defeating enemies, and reaching checkpoints. The project was developed as part of a university game development project using Agile Scrum practices.

## Features

### Core Features

* Player movement and jumping
* Dash attack combat system
* Enemy AI behaviour
* Ranged enemy system with projectile attacks
* Health and damage system
* Checkpoint and respawn system
* Multiple playable levels

### Additional Features

* Pause menu
* Game over screen
* Character and enemy animations
* Sound effects for gameplay actions
* UI health display system

## How to Build and Run

### Running the Project

1. Clone the repository from GitHub.
2. Open Unity Hub.
3. Add the project folder.
4. Open the project in Unity.
5. Open the main scene.
6. Press the Play button to run the game.

### Creating a Build

1. Open the project in Unity.
2. Select File → Build Profiles.
3. Choose the target platform.
4. Click Build.
5. Select an output folder.

## Controls

| Action      | Control           |

## Project Structure

* Assets/Scripts – Game scripts
* Assets/Scenes – Unity scenes
* Assets/Animations – Character and enemy animations
* Assets/Prefabs – Reusable game objects
* Assets/Audio – Sound effects and audio assets
* Assets/UI – User interface elements

## Scrum Development

The project was developed using Scrum methodologies across multiple sprints. Features were planned, implemented, reviewed, and improved through sprint reviews and retrospectives. GitHub was used for version control and collaboration between team members.
